## Building and Running Development environment (DEV)

* ### BUILD
  
  ````commandline  
  docker-compose build --no-cache
  ````

* ### RUN
  
  ````commandline
  docker-compose up -d
  ````

* ### STOP
  
  ````commandline
  docker-compose down -v
  ````

* ### OPEN <http://localhost:8000/>

* ### SUPER USER & DJANGO ADMIN
